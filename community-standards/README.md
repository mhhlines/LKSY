# lksy.org Community Standards

This repository contains community-maintained QA standards lists for go-to-market teams.

## Repository Structure

```
community-standards/
├── lists/                    # All approved lists (JSON files)
├── proposals/                # Proposal documents
│   ├── new-lists/           # Proposals for new lists
│   └── modifications/       # Proposals to modify existing lists
├── templates/               # Templates for lists and proposals
├── .github/
│   └── workflows/          # GitHub Actions workflows
├── schema.json             # JSON schema for validation
└── CHANGELOG.md            # Auto-generated changelog
```

## Contributing

See the main [lksy.org](https://lksy.org) website to propose new lists or modifications.
